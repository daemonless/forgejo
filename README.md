## Deployment

### Podman Compose

```yaml
services:
  forgejo:
    image: ghcr.io/daemonless/forgejo:latest
    container_name: forgejo
    environment:
      - PUID=1000
      - PGID=1000
      - TZ=UTC
      - SSH_PORT=22
    volumes:
      - "/path/to/containers/forgejo:/config"
    ports:
      - 3000:3000
      - 2222:22
    restart: unless-stopped
```

### Podman CLI

```bash
podman run -d --name forgejo \
  -p 3000:3000 \
  -p 2222:22 \
  -e PUID=1000 \
  -e PGID=1000 \
  -e TZ=UTC \
  -e SSH_PORT=22 \
  -v /path/to/containers/forgejo:/config \
```
